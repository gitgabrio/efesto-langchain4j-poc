LC4J Module
===============

This module show the usage of Efesto framework to embed a LLM provider - lc4j in this case.
It features the [Ollama](https://ollama.com/) LLM server, so it need a running instance of it in the environment.

To check ths status of the server, simply issue

```bash
ollama show llama3.2
```

To fire an interactive shell with it, issue
```bash
ollama run llama3.2
```
